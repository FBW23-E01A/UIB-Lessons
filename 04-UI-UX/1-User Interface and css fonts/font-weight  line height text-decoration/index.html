CSS font weight

CSS Font-Weight The weight of the font refers to the thickness of the font i.e. how thick or thin the font will be on the webpage. The font-weight property of CSS is used to set the weight/thickness/boldness of the font. It is either dependent on the browserâ€™s specified weights or the available font faces in a font family. It can have absolute or relative or numeric values like normal, bold, lighter, bolder, 100, 400. We do not need to specify any unit while using the numeric values

CSS line height

Line height can be thought the distance between lines of text on a webpage. The line-height property in CSS allows us to define the height of the line box, i.e., the amount of space above and below the inline elements. It accepts the values such as normal, number, length, percentage, initial.
